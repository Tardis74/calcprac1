## Задание 1

#1.1 Матрицы общего вида

Для генерации тестовых данных запустить (аргументом указать размер матрицы)
```bash
python3 generate.py 10000
```
и ждать...

После генерации запустить
```bash
make run
```
и ждать...

После выполнения программы можно запустить проверку
```bash
python3 verify.py
```
и ждать...

PS: да, генерировать данные питоном и проверять их не самая лучшая идея, но делать это паскалем ничуть не лучше

#1.2 Трехдиагональные матрицы

Для запуска программы есть несколько способов:

если надо сгенерировать данные, запустить программу и проверить средствами фортрана
```bash
make test N=10000000
```
если надо сгенерировать данные, запустить программу но не проверять правильность (для больших матриц функция проверки попытается выделить 700 терабайт оперативной памяти и будет сильно ругаться
```bash
make benchmark N=10000000
```

просто запустить программу на имеющихся тестовых данных
```bash
make run
```
